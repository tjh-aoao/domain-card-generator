# 域·卡牌生成器 (Spirit Card Generator) - 交接文档

这是一份关于“域·卡牌生成器”项目的交接文件包说明书。

## 项目概述
本项目是一个基于 React 的 Web 应用程序，专为生成、编辑和管理《域》定制卡牌而设计。它具备高保真的卡牌编辑器、支持所见即所得的预览功能、离线牌库管理功能以及高质量的图片导出（含批量导出）功能。

## 技术栈
*   **前端:** React 19, TypeScript
*   **构建工具:** Vite 6
*   **样式:** Tailwind CSS v4
*   **动画与交互:** Motion (Framer Motion), Lucide React (图标)
*   **截图与导出:** html2canvas
*   **后端服务:** Express (用于处理由于开发环境需要的静态文件和基础路由，主打全栈结构或快速部署)

## 核心功能结构
本项目核心代码主要位于 `src/` 目录下：

*   **`src/App.tsx`**: 应用的组件入口和核心主控。集成了：
    *   **卡牌编辑器 (`CardPreview`):** 负责卡牌的高保真渲染，处理图片的缩放(`imageScale`)和偏移(`imageOffset`)，最近已经修复了图片预览与导出不一致的问题。
    *   **左侧菜单/表单:** 处理数据绑定和属性更改（名称、编号、立绘、数值、背景描述等）。
    *   **牌库系统:** 将设计好的卡片存入浏览器 `localStorage` 中。支持新建卡片、克隆副本、导入/导出 JSON 牌库格式。
    *   **高品质导出功能:** 提供单个或批量的高清卡牌 PNG 导出功能。目前采用离屏(Off-screen)且不受 CSS Transform (Scale) 干扰的方式来进行安全的重绘，已解决左上角裁切问题。

*   **`src/types.ts`**: TypeScript 类型及初始数据定义（例如 `CardData`, `SavedCard`, 素材配置等）。

*   **`server.ts`**: 用于提供服务和快速部署 (配合 Vite 中间件及由于部分组件的需要采用 Express Node.js)。

## 最近更新及修复说明

1.  **牌库功能扩充:** 增加“我的牌库”页面。用户可以将设计好的卡牌保存为记录，支持加载修改、克隆以及牌库数据的 JSON 倒入/导出，数据持久化到 LocalStorage。 
2.  **批量导出:** 牌库页面新增“批量导出”功能，一次点击即可将牌库里的所有卡自动渲染并保存为 PNG 图片，使用屏幕外(Off-screen)画板规避了视图裁切问题。
3.  **渲染误差修复:** 修复了图片插图通过鼠标滚轮或拖拽后，由于坐标系及导出画布缩放 (scale) 等因素导致的“导出图片左上标题被裁切”及“导出的图片显示有问题，和预览的不一致”的问题。统一了缩放位移的状态 (`imageScale`, `imageOffset`)，并在卡片外层统一封装了离屏画板用于安全的 `html2canvas` 导出。

## 本地开发与部署指南

如果您需要在本地运行此项目，请确保您安装了 `Node.js` (推荐 v18+)。

```bash
# 1. 安装依赖包
npm install

# 2. 启动开发服务器（含热更新）
npm run dev
# 运行后在浏览器访问 http://localhost:3000

# 3. 构建生产环境代码
npm run build

# 4. 运行生产环境版
npm run start
```
